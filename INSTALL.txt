### Install
## Requisitos del Sistema.
-PC/Laptop/Notebook con Sistema operativo Windows.
-Compilador C++.
-Se debe contar con el programa DEVC++ en su versión 5.11.
# Proceso de Instalación
- Clonar el repositorio en su PC/Laptop/Notebook de preferencia.
- bash
- Copy code
- git clone https://github.com/Ranafonica/Proyecto-3/blob/master/Proyecto%203.cpp (de preferencia)
Abrir el archivo .cpp con el programa DEVC++ 5.11.
En la barra de herramientas ubicada en la zona superior, hacer clic en el botón "Compilar y Ejecutar" o pulsar "F11" en su teclado.
Nota: Si su PC/Laptop/Notebook tiene problemas al compilar debido a la versión de su ISO, siga uno de los siguientes procedimientos:
Forma A: Project -> Project Options -> Compiler -> Code Generation -> Language standard (-std). Y finalmente, establezca: GNU C++11.
Forma B: Compiler options -> Settings -> Code generation -> Language standard y elija GNU C++11.
# Ejecución del Programa
- Se abrirá una pequeña ventana en negro.
- Siga las instrucciones en pantalla.
